# EmuWheel-RE

EmuWheel-RE contains decompiled (reverse-engineered) versions of [Forza EmuWheel](https://forzatools.weebly.com/forza-emuwheel.html), originally by FirstPlatoLV, made with FirstPlatoLV's permission as the original source is unavailable.

This repository contains the decompiled source code of the main EmUWheel app, with additional bugfixes and improvements.

## How to use

- Download [Forza EmuWheel](https://forzatools.weebly.com/forza-emuwheel.html)
- Replace `Forza EmuWheel.exe` with the version from [the latest release](https://github.com/EmuWheel-RE/Forza EmuWheel/releases/latest)
- Follow [the EmuWheel Setup Guide](https://forzatools.weebly.com/forza-emuwheel-setup-guide.html)
